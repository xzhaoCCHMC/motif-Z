Input format

Input sequences should be in one-letter amino acid code (not case sensitive), with one peptide in each row and without headers. Input sequences can contain unknown or unspecified amino acid, which should be denoted as "X". Peptide sequences with incorrect format will be ignored. Format of the example set in the webpage can be used as a reference. Maximum peptides is 9,999 peptides per submission for reasonable usage of computational resources at a time. {% endblock %}
