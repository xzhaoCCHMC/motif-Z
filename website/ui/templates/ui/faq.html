What is motif-Z designed to do?

Motif-Z was designed to identify motifs from large tandem mass spectrometry-based data sets of pMHC class I. Since the algorithm has no discrimination on input peptides, Motif-Z is able to detect motifs from practically any sequence based data set - either pre-aligned or with different lengths. Currently it supports datasets with length from 7 to 14 amino acids.

How to upload input sequences on motif-Z?

You can upload input sequences either by selecting a file on your local disk or by cut and paste your sequences in text format. The file must be in either csv or txt files. If the input is uploaded from your local disk, corresponding checkbox indicating file format should be checked.

How to run motif-Z?

Most of parameters in motif-Z were set with default values. These parameters were selected and validated during developement with testing datasets, so users can use these setting without change. After clicking on the "Submit" button, the status of your job will be displayed and constantly updated until it terminates and the server output appears in the browser window.

What is binomial probability threshold?

The threshold of binomial probability to be considered significant in motif identification. 0.000001 is a stringent value as threshold. Binomial probability threshold will only be used when you don't use Z-Score, which is recommended in the computation of motif due to its statistical power. It can be leave as blank, which indicates using Z-Score and ignores this threshold.

What is maximum peptide length?

Maximum peptide length is the longest peptide in the input data will be analyzed by motif-Z algorithm, default value is 14.

What is minimum peptide length?

Minimum peptide length is the shortest peptide in the input data will be analyzed by motif-Z algorithm, default value is 7.

What is minimum occurrence in input dataset threshold?

The threshold used to filter the motifs having minimum coverage in the input dataset. The default value is 20 which means that identified motif has at least 20 peptides in the input data possessing this motif. Setting this parameter usually depends on your dataset size. Changing this parameter to a larger number will affect the depth of recursive search and result in motifs with less residue position pairs generally.

What is peptide length parameter in pre-aligned mode?

This parameter only used for datasets with pre-aligned peptides with same length. The default value is 9.

What is background dataset used in the motif-Z?

Background peptide sequences were generated from a database of known human protein sequences (IPI Human v3.66) using a Markov chain method to simulate peptides randomly generated from human proteins. The range of peptide length is from 7 to 14 amino acids with a total number of 1 million peptides.

Is there any test data set that I could try out?

You could try a experimental tandem mass spectrometry data set, about 1000 peptide sequences, from B-LCL-HHC cell line. Download it here. Then upload it onto motif-Z either by cutting and pasting the sequences or by uploading the actual file. You can just use default parameters. If you did everything correctly you should get back 28 motifs.

What is the confidentiality of my data submitted to motif-Z?

All sequences submitted to motif-Z are kept confidential and will be deleted after processing. Your results will be kept in 48 hours in our server for your download and will be deleted after that.
