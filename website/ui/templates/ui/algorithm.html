The algorithm of motif-Z was based on motif-X with significant change and reimplementation. It used an iterative strategy which builds successive motifs through comparison to a dynamic statistical background. Motif-Z adopted Z-Score for significant residue/position pair determination and used C,N-termini alignments for peptides with various lengths. The detail of the algorithm is described in our paper: A sensitive, length-independent algorithm for identifying motifs from MHC class I peptide ligands.
