Minimum occurrence in input dataset

The threshold used to filter the motifs having minimum coverage in the input dataset. The default value is 20 which means that identified motif has at least 20 peptides in the input data possessing this motif. Setting this parameter usually depends on your dataset size. Changing this parameter to a larger number will affect the depth of recursive search and result in motifs with less residue position pairs generally. {% endblock %}
