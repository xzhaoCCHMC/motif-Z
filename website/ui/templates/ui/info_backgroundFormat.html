Upload background peptide sequence file

To run Motif-Z, you can either use our provided background files or upload your own background for motif searching. Default option is "Uniprot human peptide" inlcuding 1 million peptide sequences generated with various length using proteins from Uniprot database generated following HMM model. Option "Random peptides" inlcudes 1 million peptides with random distribution. For uploaded files, currently only text file will be accepted (format as in example, i.e. one peptide per line). The size of the uploaded file should be less than 10 MB, about 1 million peptides. Bigger size and other formats won't be accepted at this time. {% endblock %}
